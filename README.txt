GaiaSim

GaiaSim is a visual simulation of continental interaction made in Unity. It aims to assist world-builders in creating fantasy maps.

The Windows version of the software may simply be run by executing GaiaSim.exe

- Begin the simulation by pressing the Play button.
- The simulation can be paused and resumed with this same button.
- Controls for altering the behaviour of the simulation can be found on the right hand side of the screen while paused.
- Using the Save Map button will save a PNG heightmap image for the current map state in the same folder that the program was executed in.
